Recommended usage

Spectra is mobile-first. Start from the base layout, then widen at the point where the composition genuinely needs more room.

/* --sp-bp-md */
@media (min-width: 52rem) {
  .docs-grid {
    grid-template-columns: repeat(2, minmax(0, 1fr));
  }
}
