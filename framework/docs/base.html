Base

The base layer provides a minimal, predictable baseline before components. It should stay small and token-driven.

Start Here · Getting Started · Tokens · Components · Layouts · Versioning

Reset

src/base/reset.css sets:

  • box-sizing
  • removes default margins
  • token-driven body defaults (bg/text/font/size/line-height)
  • predictable media sizing
  • form elements inherit typography

A11y focus

src/base/a11y.css defines a consistent :focus-visible ring using: --sp-outline-1, --sp-ring-1, --sp-focus, --sp-focus-soft.

Layout helpers

src/base/layout.css provides a few tiny primitives:

  • .sp-container — centered container with max width + token padding
  • .sp-content — narrower max width for readable docs content
  • .sp-stack — simple vertical spacing between children

Import order

Base is imported from src/spectra.css after tokens.
