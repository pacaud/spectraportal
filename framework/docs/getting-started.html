Getting Started

The goal of this page is simple: get your first success in a few minutes, then point you to the next step.

Start Here · Getting Started · Tokens · Components · Layouts · Versioning

1) Install (CDN)

Add Spectra to any HTML page. No build tools required. 

<link rel="stylesheet" href="https://spectraportal.dev/framework/cdn/v0.1/spectra.css" />

2) Drop in something visible

<button class="sp-button sp-primary">Hello Spectra</button>

If you see a nice button with clean spacing, you’re set.

3) Change one token (optional, but satisfying)

Override tokens in your own CSS file after Spectra. Start with one variable: 

:root{
  --sp-primary: #7c5cff;
}
