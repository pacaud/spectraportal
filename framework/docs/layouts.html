Layouts & Patterns

Spectra stays lightweight, so “layout” lives mostly in a few tiny primitives. This page shows common patterns you can build with them.

Start Here · Getting Started · Tokens · Components · Layouts · Versioning

The three primitives

  • .sp-container — centered container with max-width + token padding
  • .sp-content — readable content width (used by docs)
  • .sp-stack — vertical spacing between children (no custom margins)

Pattern: readable page

<main class="sp-container sp-content">
  <section class="sp-stack">
    <h1>Title</h1>
    <p>Body text…</p>
  </section>
</main>

Pattern: card grid

Use your own grid (CSS grid/flex) and place Spectra surfaces inside.

<div style="display:grid; gap:var(--sp-4); grid-template-columns:repeat(auto-fit,minmax(16rem,1fr));">
  <article class="sp-card">...</article>
  <article class="sp-card">...</article>
</div>

Pattern: tool layout (sidebar + content)

Spectra won’t force a layout system — but it plays nicely with simple flex patterns.

<div style="display:flex; gap:var(--sp-4); align-items:stretch;">
  <aside class="sp-card" style="width:18rem;">Sidebar…</aside>
  <section class="sp-card" style="flex:1;">Content…</section>
</div>
