# Buttons & Actions

Spectra action elements are designed to feel consistent across tools and pages.

## Common classes

- `sp-button` — base button styling
- `sp-primary` — primary (accent) action
- `sp-secondary` — secondary action
- `sp-ghost` — low-emphasis action

## Quick example

```html
<button class="sp-button sp-primary">Primary</button>
<button class="sp-button sp-secondary">Secondary</button>
<button class="sp-button sp-ghost">Ghost</button>
```

## Notes

- Use tokens for sizing/spacing in any custom variants.
- Keep the number of button styles small; consistency beats options.
