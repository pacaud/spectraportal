Tokens

Tokens are variables that define Spectra’s entire design system. Components and base layers should only consume var(--sp-*).

Naming

  • --sp-* prefix always.
  • Use clear groups: --sp-radius-*, --sp-shadow-*, --sp-duration-*, etc.
  • Prefer small, consistent scales over lots of one-off values.

Quick edit example

Tokens are just CSS variables. Override them in your own stylesheet after Spectra loads: 

:root{
  /* Try changing one thing first */
  --sp-primary: #7c5cff;
}

Breakpoint note (token-ish)

For compatibility, do not use var() inside @media. Use the literal value and reference the token name in a comment. 

/* --sp-bp-sm (40rem) */
@media (min-width: 40rem){ 
  /* rules */
}
