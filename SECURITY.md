# Security Policy

If you believe you’ve found a security vulnerability, please **do not** open a public issue.

## Report a vulnerability
Email: security@spectraportal.dev

Include:
- A clear description of the issue
- Steps to reproduce (if possible)
- Impact / worst-case scenario
- Any suggested fix or mitigation
- The commit hash or release/tag you tested

We’ll respond as soon as we can and coordinate a responsible disclosure timeline.
